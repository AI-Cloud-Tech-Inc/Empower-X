# AI/ML Debugger Extension Setup Instructions

## 1. Install the Extension
- Open VS Code.
- Go to Extensions (Ctrl+Shift+X).
- Search for "AI/ML Debugger".
- Click "Install" on AI/ML Debugger v1.7.1.

## 2. Basic Configuration
- After installation, open the AI/ML Debugger Dashboard from the sidebar.
- Click "Auto-Detect" to let the extension identify your ML framework.
- Optionally, configure settings in `.vscode/settings.json` for custom paths or framework preferences.

## 3. Usage
- Use "Quick Start" for guided setup.
- Use "Data Health" to analyze your dataset.
- Use "Optimizer Wizard" for model optimization.

## 4. Sample Python/ML File
See `sample_ml_debug.py` for a demonstration of extension features.

## 5. Integration
- Open your ML script in VS Code.
- Use the debugger panel to set breakpoints, inspect tensors, and analyze metrics.
- Refer to the extension documentation for advanced features.
